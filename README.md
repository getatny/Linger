## 如何运行

该小程序使用 [Tarojs]( https://taro.aotu.io/ ) 开发，所以首先你得确保你电脑已经安装了`Node`。你可以使用`node -v`查看本机 Nodejs 版本，确定已安装Node之后，运行：

```bash
npm install && npm run dev:weapp
```

当然如果你使用yarn作为包管理器，也可以这样：

```bash
yarn install && yarn dev:weapp
```
